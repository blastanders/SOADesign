OVERVIEW

Las Vegas-based nonprofit Keep Memory Alive is passionate about increasing awareness and raising funds for the research, management, and treatment of brain disorders.
Our advertising agency was selected to design and develop Keep Memory Alive's new website, which features custom video, photography, a virtual facility tour, an event registration system, and a donation system that allows users to easily support Cleveland Clinic Lou Ruvo Center for Brain Health.
